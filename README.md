##### 启动hugo服务器
```shell
hugo server
```
##### 启动hugo(包含草稿内容)
```shell
hugo server --buildDrafts

hugo server -D

hugo server -D --disableFastRender
```

```shell
hugo new content posts/xxx.md
```
